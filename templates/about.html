About SecQR

SecQR is a web based application that lets you check QR codes for dangerous URLs and send secret messages with QR codes. It supports image uploads and direct QR code scanning.

To send a secret message via a QR code, simply encode your custom QR code with a normal message and a secret message. Then the recipient can use SecQR to decode the normal message and the secret message.

The secret message remains hidden for any normal scanner, it can only be decoded using SecQR. Even further, before being encoded int–æ the QR code, the secret message is encrypted, providing maximum level of data confidentiality. Note that there are some limitations - the normal message can be maximum 84 characters long and the secret message can be maximum 57 characters long.

Enjoy SecQR!
